1. set() 집합을 의미하는 set!
- 반복가능한 객체를 넣어서 만들수 있음 하지만 겹치는건 하나로 처리되고 앞에 있는 게 나오는 대신 뒤에 것이 출력됨!
- 숫자는 그대로 들어감, 하지만 문자열만 다름!
- 인덱스를 가지고 오지 못하는 이유는 순서가 없기 때문


2. set.union(세트1, 세트2)
   - 세트 | 세트 => <합집합> .....우리들
   - 세트 |= 세트 => 세트.update({}) -> 원하는 세트 추가하기!
  
     
3. set.intersection(세트1, 세트2)
   - 세트&세트2 => <교집합>.....우리가 겹치는 거
   - 세트&=세트 => 세트.intersection_update({})-> 겹치는 애들 업테이트해!


4. set.difference(세트 1, 세트 )
  - 세트 - 세트 => <차집합>....먼저 쓴것만 해당되는거! 걍 only 자기자신
  - 세트 -= 세트 = 세트.difference_update({})-> 나만 업데이트

 
5. set.symmetric_difference(세트1, 세트2)
  - 세트 ^ 세트 => <대칭 차집합>.... 겹치는거 빼고 구하기
  - 세트 ^= 세트 => 세트.symmetric_difference_update({})-> 겹치는 거 빼고 업데이트 해주라

6. 현재 세트 <= 다른 세트 = 현재시트.issubset({}) -> 우리가<부분집합>이니?
   - 둘다 같은 것
   - (ex){1,2,3}, {1,2,3}끼리하면 불(bool)값이 나온다.
  

7. 현재 세트 < 다른 세트 = 이것도 issubset에 합쳐진것이라 메서드가 없당.
   - 부분집합이라, 같으면 false가 나옴 !
   - 부등호에 따라서 검출이 가능함!
   - (ex){1,2,3,4}<{1,2,3,4} false !


8. 현재 세트 >= 다른 세트 = 현재시트.issuperset({}) -> 내가 <상위 집합>이나?
   - 서브랑 반대로 더 큰 형태이다 !!
   - (ex){1,2,3,4}랑 {1,2,3,4} true / {1,2,3,4}랑 {1,2,3} True


9. 현재 세트 > 다른 세트 = 얘도 메서드가 없당~ 다른 세트의 진상위집합인지 묻는 거라
    - {1,2,3,4}랑 {1,2,3} false !
